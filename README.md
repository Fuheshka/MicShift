**MicShift** — это современная системная утилита для Windows, написанная на C# (.NET 10 & WPF), которая позволяет легко переключаться между микрофонами, настраивать глобальные горячие клавиши для управления звуком и автоматически переключать активное устройство записи в зависимости от того, какое из них сейчас ближе к вам (технология **Auto-Distance**).

---

## ✨ Основные возможности

* 🎙️ **Автоматическое переключение (Auto-Distance):** 
  Приложение в реальном времени анализирует уровень звука с двух ваших микрофонов (например, настольного конденсаторного и встроенного в гарнитуру).
  * **Защита от ложных переключений (Hysteresis):** Переключение происходит только в том случае, если уровень звука превышает порог тишины ($> 3\%$) и стабильно держится выше другого микрофона в $1.5$ раза на протяжении $\approx 200$ мс (6 циклов опроса). Это предотвращает «дребезг» при одинаковом уровне шума.
* 🖥️ **Красивый темный GUI (WPF):** 
  Современная панель управления в темных тонах с плавными индикаторами уровня сигнала (VU-метрами) для каждого из микрофонов, удобным выбором калибруемых устройств и простым тумблером активации.
* 📦 **Скрытая работа в трее (System Tray):**
  * Приложение может работать в фоновом режиме. Окно полностью скрывается, оставляя иконку в системном трее.
  * **Восстановление в один клик:** Двойной клик на иконку в трее или пункт меню **«Open Settings»** мгновенно возвращает интерфейс на экран.
* ⌨️ **Глобальные горячие клавиши (WinAPI):**
  Работают в любой игре или полноэкранном приложении:
  * `Ctrl + Alt + M` — включить / выключить звук активного микрофона (Mute/Unmute).
  * `Ctrl + Alt + S` — переключить микрофон по умолчанию на следующее активное устройство по кругу (Cycle).
* 🔔 **Всплывающие Toast-уведомления:**
  Нативные уведомления Windows сообщают о смене микрофона или его заглушении.
* ⚙️ **Полная поддержка CLI:**
  Возможность интеграции в скрипты автоматизации или панели управления вроде Stream Deck.

## 🛠️ Параметры командной строки (CLI)

Если приложение запускается с аргументами командной строки, оно автоматически подключается к вашей консоли (PowerShell/CMD), выводит результат и мгновенно завершает работу без открытия графического интерфейса:

```bash
# Показать список всех активных микрофонов с их GUID-идентификаторами
MicShift.exe --list

# Показать текущее устройство связи по умолчанию
MicShift.exe --default

# Переключить микрофон по его названию (или части названия) или по GUID
MicShift.exe --switch "HyperX SoloCast"
MicShift.exe --switch "00000000-0000-0000-0000-000000000000"

# Запустить приложение скрытым непосредственно в системный трей
MicShift.exe --tray

# Вызвать справку по аргументам
MicShift.exe --help
```

## 🚀 Настройка автозапуска в Windows

Чтобы MicShift запускался вместе с Windows сразу в скрытом режиме (в системном трее):
1. Нажмите `Win + R`, введите `shell:startup` и нажмите Enter (откроется папка автозагрузки).
2. Создайте ярлык для файла `MicShift.exe`.
3. Нажмите правой кнопкой мыши по созданному ярлыку -> **Свойства**.
4. В поле **«Объект»** в самом конце через пробел добавьте аргумент `--tray` (или `-t`).
   * *Пример:* `"C:\Program Files\MicShift\MicShift.exe" --tray`
5. Нажмите **ОК**. Теперь программа будет запускаться в фоне при старте системы.

## 📦 Сборка и компиляция

Для сборки оптимизированного исполняемого файла, который содержит в себе рантайм .NET и работает на любом ПК с Windows x64:

```bash
dotnet publish MicShift.csproj -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:PublishReadyToRun=true -p:PublishTrimmed=false -o ./publish
```

Скомпилированный файл `MicShift.exe` появится в папке `./publish`.
